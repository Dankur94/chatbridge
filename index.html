The Architecture

Three layers. That's it.

01

Chat Platform

Pick the messenger your users already have. WhatsApp, WeChat, Telegram, LINE — doesn't matter. It's just the interface.

02

Smart Backend

A modular engine that understands messages, routes them to the right module, stores data, and triggers automations. Regex first, AI only when needed.

03

App Integrations

Calendar, e-commerce, IoT sensors, payments, AI — plug in whatever your use case needs. Each integration is a module. Add or remove without touching the rest.

Under The Hood

Simple by design

No complex frameworks. No vendor lock-in. Python + SQLite + your preferred chat API. Every feature is a module with three methods.

modules/your_feature/module.py 
class YourFeature(BaseModule):

    def can_handle(self, message) -> bool:
        # Does this message belong to your module?
        return "keyword" in message.text.lower()

    def handle(self, message) -> Response:
        # Process it and return a reply
        self.db.save(message)
        return Response(reply="Done!")

    def get_scheduled_jobs(self) -> list:
        # Automated messages (reminders, reports)
        return [ScheduledJob(cron="0 22 * * *", func=self.daily_report)]
