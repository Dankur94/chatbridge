# BitsBridge Website

## Was ist das?
Minimale Visitenkarte / Landing Page für Daniel Kurth.
Zweck: Glaubwürdig aussehen wenn jemand den Namen googelt.

## Struktur

### index.html (Visitenkarte)
- **Nav**: Logo "BitsBridge" + About-Link + Sprach-Toggle (EN/DE)
- **Hero**: Profilfoto + "Daniel Kurth" + Ein-Zeiler + Buttons (About Me, LinkedIn)
- **Contact**: 3 Karten (Email, WhatsApp, LinkedIn)
- **Footer**: Minimal (BitsBridge, About-Link, Email, GitHub)

### about.html (Über mich)
- Ausführliche Seite über Daniel Kurth (Geschichte, Skills, Kontakt)
- Bleibt wie sie ist — guter Content

### 5 Unterseiten (nicht verlinkt, für spätere Nutzung)
- maschinenbau.html, importeure.html, fertigung.html, kmu.html, prototypen.html
- Sind von der Navigation entlinkt, bleiben als Dateien

## Design-Regeln
- Tesla-inspiriert, Full-Screen Sections, dark/light, Inter Font
- Bestehende style.css und script.js wiederverwenden
- Responsive (Mobile-First)
- Hauptseite: Englisch (lang="en"), Sprach-Toggle für EN/DE
- About + Unterseiten: Deutsch (lang="de"), Sprach-Toggle für EN/DE
